Machine Learning in IOS 11: Accelerate / Metal and Swift

Proposal

Machine Learning on IOS 11, the GPU and the CPU

Apple had recently presented a native machine learning (CoreML) and machine vision (Vision) frameworks augmenting the already existing libraries they provided (e.g, Accelerate). This significantly enhances the vast array of ML based apps that can be deployed to IOS based devices.

In this session, an offline trained Keras ML model will be deployed to an iOS target wherein an application will subsequently consume the ML model using a basic Swift based UI.

The model’s learned parameters are loaded into the app in run-time and subsequently, in order to make make a prediction, the app will run the inference locally on IOS, using its own CPU or GPU.

Stack:

  • CoreML
  • Accelerate
  • Metal
  • Swift
  • iOS 11.0 Beta
  • Xcode 9+
